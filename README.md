# Monstera Clicker

Hi from 2018.

This is a small clicker game and one of the very first games I ever made. The original version was built years ago as a simple browser experiment: click, grow the plant, unlock progress, collect coins, and find a few hidden things along the way.

This repository is a separate refreshed version, cleaned up and brought back to life in 2026. The goal is to keep the old handmade feeling, but give the project a more modern structure so it can be improved without losing its original charm.

## Development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
